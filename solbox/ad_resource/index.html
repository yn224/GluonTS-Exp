Introduction

I was given a task to look through different anomaly detection algorithms that could potentially be implemented later in real-world setting. Throughout my internship period, I managed to learn about various algorithms and pros and cons of them with respect to others. The algorithms range from those that have statistical background to Neural Networks. Main source of knowledge was through number of papers dedicated to the introduction and analysis of such algorithms.

Starting Point

I believe this Github repository contains enough learning resources for anyone trying to learn outlier (anomaly) detection methods. Though the time series anomaly detection is not the focus, the content still contains some information regarding anomaly detection in time series data.
